# docs

프로젝트 루트 README와 서브 README에 다 담기지 않는 보조 문서, 세션 인수인계용 메모, 작업 명세 파일을 두는 디렉터리입니다.

## 현재 용도

- 세션 인수인계용 보조 문서
- 서브 세션용 작업 명세
- 임시 실험 문서

## 주의사항

- 루트 `README.md`와 충돌하지 않게, 보조 성격의 문서만 두는 쪽이 좋습니다.
- 장기 유지가 필요한 문서는 가능하면
  - 루트 `README.md`
  - `backend/README.md`
  - `frontend/README.md`
  - `collector/README.md`
  쪽으로 승격하는 편이 낫습니다.
